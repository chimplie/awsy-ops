Chops (Chimplie Ops)
====================

Basic DevOps toolset for AWS and Docker we use in Chimplie

Installation
------------

```bash
pip install git+https://github.com/chimplie/chops.git
```

Usage
-----

To use programmatically, simply do:

```python
import chops
chops.program.run()
```

Development
-----------

To install development version of the plugin:

```bash
pip install -e .
```
